# Emotion Detector

A web-based emotion detection application built using IBM Watson NLP Library and Flask.

## Project Overview

This application analyzes input text and detects emotions such as joy, sadness, anger, disgust, and fear using the Watson NLP Emotion Predict API. The dominant emotion is also returned.

## Tasks Completed

- Task 1: Clone the project repository
- Task 2: Create an emotion detection application using the Watson NLP library
- Task 3: Format the output of the application
- Task 4: Package the application as EmotionDetection
- Task 5: Run Unit tests on the application
- Task 6: Web deployment of the application using Flask
- Task 7: Incorporate error handling
- Task 8: Run static code analysis

## Technologies Used

- Python 3
- Flask
- IBM Watson NLP Library
- Requests

## Running the Application

```bash
python server.py
```

Then open your browser and go to `http://localhost:5000`
